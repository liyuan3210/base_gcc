	1).缓冲文件系统
	系统自动为文件开辟一个缓冲区

3>.读写方式
	打开文件：fopen
	关闭文件：fclose
	1).fputc,fgetc字符读写。
	每次只能读写一个字符。
	2).fread,fwrite结构数据，数据组读写。
	通常用来读写一组数据或结构体数据
	3).fprintf,fscanf格式化读写
	用来格式化读写数据的，与printf函数类似
	4).putw,getw
	5).fputs,fgets字符串读写
	通常读写字符串
	
4>.文件定位
	1.rewind(文件指针):使指针重新返回文件头的指针
	2.fseek(文件指针，位移量，起始点):改变文件位子指针
	3.ftell(文件指针):用于得到流式文件的当前位置，如果函数返回-1L表示出错
